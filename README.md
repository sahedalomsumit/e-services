# E-Services Website

Static multi-page website for **E-Service Parts**, focused on digital signage, kiosk, gaming, and vending machine parts.

This repository contains standalone HTML pages for key storefront and support flows such as product browsing, cart/checkout, account pages, policies, and contact/help content.

## Project Type

- Frontend-only static site
- Pure HTML with inline CSS/JS in page files
- No build step or package manager required

## Main Pages

- `homepage-final.html` - Homepage and storefront landing
- `shop.html` - Product listing
- `single-product.html` - Product detail page
- `cart.html` - Shopping cart
- `checkout.html` - Checkout flow
- `thank-you.html` - Order confirmation
- `my-account.html`, `trade-account.html`, `lost-password.html` - Account pages
- `mini-cart.html`, `wishlist.html`, `track-order.html` - Customer utility pages
- `about-final.html`, `contact-us.html`, `faq-final.html` - Company and support pages
- `privacy-policy.html`, `cookie-policy.html`, `terms-and-conditions.html`, `return-policy.html` - Policy/legal pages

## Getting Started

1. Clone or download this repository.
2. Open `homepage-final.html` in your browser.
3. Navigate through links/pages to review the full flow.

## Notes

- This project is suitable for static hosting (for example: GitHub Pages, Netlify, Vercel static output).
- If you want shared reusable styles/scripts later, pages can be refactored to external `css` and `js` assets.
